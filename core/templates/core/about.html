Acerca de este proyecto

Este blog fue desarrollado como Proyecto Final utilizando Django, aplicando autenticación, gestión de usuarios, publicaciones con imágenes y categorías.

El objetivo es demostrar el uso del framework Django siguiendo el patrón MVC, con buenas prácticas y diseño responsive.

  • Registro y login de usuarios
  • Roles (admin / colaborador)
  • CRUD de publicaciones
  • Categorías y filtrado
  • Subida de imágenes
